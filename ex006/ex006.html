Listas em Html.

Explicando como aplicar a tag <Ol> (ORDERED LIST)

Para aplicar a lista ordenada deve-se chamar as tags de abertura e fechamento <ol></ol>, entre elas devemos colocar os itens da lista chamando a tag <li></li> (list item), vamos a um exemplo.

EX:.. 

            <ol>
            <li> 1 items</li>
            <li> 2 items</li>
            <li> 3 items</li>
            </ol>

A tag <ol> possui um parâmetro type onde configuramos o tipo de marcado, esses marcadores podem se 1,2,3.., A,B,C.., a,b,c.., I,II,III,IV..,por padrão os itens desta lista seram numerados, podemos também defenir em que inico de marcador os itens da lista podem comecar, com o parâmetro start.

EX:. 



                <ol type="I" start="3">

                <li> III items </li>
                <li> IV items </li>
                <li> V items </li>

                </ol>

Listas não ordenadas <ul> (unordered list).

A aplicação das tags não ordenadas,<ul></ul>, como mostrada tem abertura e fechamento, entre elas aplicamos a tag <li></li> para adicionar os items da lista não ordenada.

EX:. 

            <ul>
            <li> item </li>
            <li> item </li>
            <li> item </li>
            </ul>

Nas listas não ordenadas tembém podemos mudar os marcadores podendo ter até 3 tipos, sendo disc (uma bola preta toda pintada,essa por padrão, circle um círculo com uma borda preta sem preenchimento e square um quadrado preto todo pintado, segue o exemplo a baixo:

EX:.

  • Disc
  • Circle
  • Square

Lista mista

Em uma lista mista podemos colocar lista em cadeia, ou seja uma dento de outra isso ordenada com ordenadas ou vice versa, abaixo segue o exemplo.

Ex:. 

    <ol>
      <li>Item </li>
      <li>Item </li>
      <li>Item </li>
         <ol>
            <li>Item </li>         
            <li>Item </li>      
            <li>Item </li>
         </ol>
      <li>Item </li>  
      <li>Item </li>
      <li>Item </li>
    </ol>
